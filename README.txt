Utility to check if online or offline. Install and just type `online` in the Witchcraft/Terminal
 
Usage:
 
if online -s; then
   echo "✅ Its online"
   else
      echo "❌ Its offline"
fi

Installing:

Open terminal on the clonned repo folder, and type following by pressing `[ENTER]`:

chmod +x install.sh && sh ./install.sh
